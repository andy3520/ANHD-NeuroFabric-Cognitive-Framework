# Contributing to ANHD-NeuroFabric

Thank you for your interest in the ANHD-NeuroFabric Cognitive Framework!

This project is exploratory and community-driven. Contributions of all kinds are welcome:

## Ways to contribute

- Implement prototype “cells” (agents)
- Improve Coordinator / Analyst / Critic logic
- Add diagrams, theory, benchmarks, or references
- Fix bugs or improve structure
- Provide architectural critique or design proposals

## How to contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Open a pull request with a clear description

Please follow the license (CC BY-NC 4.0) and credit the author.

## Code of conduct

Be respectful, constructive, and curious.
This project explores synthetic cognition — experimentation and open minds are essential.
